## Repo Structure

Each mockup or set of mockups is a self-contained HTML file. The root `index.html` is whatever the current active mockup is. Older mockups can be archived under `/archive/` once superseded.

## Privacy

All pages include `noindex, nofollow` meta tags so search engines do not surface these in results. The repo itself is public, but mockup content is not search-indexed.
